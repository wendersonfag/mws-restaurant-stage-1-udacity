# Restaurant Reviews App (Stage 1)

## Visão do projeto
Para os projetos do Restaurant Reviews , foi convertido, uma página da Web estática em um aplicativo da Web preparado para dispositivos móveis.  Para ter um design estático que não tem acessibilidade e converterá o design para ser responsivo em exibições de tamanhos diferentes e acessível para uso do leitor de tela. E também foi adicionado um ServiceWorker para iniciar o processo de criação de uma experiência offline perfeita para seus usuários.

## Para Executar a aplicação
Para ser executar a aplicação, e necessário fazer o download ou clone o repositório em seu computador:
### Na nova pasta, é necessário fazer a inicialize de um servidor cliente local, para iniciar o servidor cliente local com o Python no seu terminal: 
Python 2: `python -m SimpleHTTPServer 8000` 
Python 3: `python3 -m http.server 8000`

### Para Acessar o site no seu navegador:
Vá para http://localhost:8000 ou em outra porta, desde que seja especificado na hora da criação do servidor local.
